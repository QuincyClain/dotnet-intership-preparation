# dotnet-intership-preparation

Учебный репозиторий для повторения C# и .NET.

Здесь я собираю практические задания, небольшие эксперименты и заметки по базовым темам: синтаксис C#, коллекции, методы, делегаты, LINQ и другие вещи, которые важно уверенно понимать на уровне .NET developer.

## Что внутри

- `Program.cs` - точка входа консольного приложения.
- `Tasks.cs` - практические задания и примеры по темам C#.
- `SyntaxTest.csproj` - проект .NET 8.
- `SyntaxTest.sln` - solution-файл для удобной работы в IDE.

## Завершенные темы:

- базовый синтаксис C#;
- условия и циклы;
- массивы и коллекции;
- `List<T>` и `Dictionary<TKey, TValue>`;
- методы;
- делегаты;
- LINQ: `Where`, `Select`, `OrderBy`, `First`, `Any`, `All`, `Count`, `Take`, `Skip`, `GroupBy`, `ToDictionary`.

## Цель

Цель репозитория - системно повторить фундамент .NET-разработки, закрепить синтаксис на практике и подготовиться к техническим вопросам на стажировку.
